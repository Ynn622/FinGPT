# FinGPT

歡迎使用 FinGPT～

FinGPT結合ChatGPT幫您分析股票！

## 使用方式
![FinGPT](https://github.com/user-attachments/assets/16ca59b9-68ba-4f75-9bc6-1936351accbc)
1. 掃描上方 QR Code 並加入好友
2. 輸入 台股股票代號
3. 按下傳送
4. 等待分析（約1分鐘 請耐心等候～）
5. 查看股市分析！

### 使用範例
* 想搜尋台積電->輸入「2330」
  * 即可查看股市分析！

### 🆕進階功能
* 如果想細部詢問 可使用換行來增加問題！
* 範例：
```
2330
想詢問止盈止損價位 該如何設定
```

## 🔔溫馨提示
* 「選單列」可查看大盤行情📈
* 目前僅提供台股「上市、上櫃」查詢！
